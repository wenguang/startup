**gcc 最开始的时候是 GNU C Compiler。但是后来因为这个项目里边集成了更多其他不同语言的编译器，GCC就代表 the GNU Compiler Collection，所以表示一堆编译器的合集。 g++则是GCC的c++编译器。** 



**C和C++编译器是集成的，编译一般分为四个步骤：**

1. **预处理(preprocessing)  ----------------- cpp/ gcc -E **
2. **编译(compilation) ------------------ cc1 / gcc -S**
3. **汇编(assembly)  -------------------- as**
4. **连接(linking) --------------------- ld **
