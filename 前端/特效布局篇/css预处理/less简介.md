## less简介

*less和sass功能类似，但它是node编写的，详细请参考 [less教程](http://www.bootcss.com/p/lesscss/)， 它支持以下特性：* 

* 变量
* 混合
* 带参数混合
* 嵌套规则
* 运算
* Color函数
* 命令空间
* 作用域
* 注释
* Importing



**安装与使用** 

> npm install less
>
> lessc -x style.less > style.css	//编译压缩

[less教程](http://www.bootcss.com/p/lesscss/) 还介绍了服务器端用node调用less与及客户端使用less的方法
